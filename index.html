Our Project

DOT bot

DOT bot-- or Department of Transportation bot-- is an all-in-one flight and travel assistance chatbot. Using information from the Department of Transportation's website, this web application can answer any questions you may have about air travel.


DOT bot uses a generative model specifically trained from a variety of questions and answers from the DOT Aviation Consumer Protection website to produce responses to user input.


For this project, we used React's Next.js framework with TailwindCSS and TypeScript to try and build a web app for our project with a modern design. Furthermore, the OpenAI API was integrated in order to train our bot.

Process

Steps of our Process

Tech-Stack

For this project, we used React's Next.js framework with TailwindCSS and TypeScript to try and build a web app for our project with a modern design. Python's chatterbot library was used in development until problems led us to switch to OpenAI's API.

Design

A lot of time was put into the design stage of our app. Color schemes, layouts, logos, and even how we wanted to present our project were up for debate.

Integration/Adaptation

"Integration Hell" is coined for a reason. This project was no exception. As our 36 hours came to a close, our improvisation and compromisation skills (and perhaps our sleep deprevation skills) were put to the test.

Challenges

Challenges Faced and Lessons Learned

Integrating python code to work in conjunction with our Next.js project was more difficult than planned. Due to last minute difficulties with deploying a work-around, we ended up switching libraries.


This worked out well for us this time, but had there not been an easy work around, we would have been in big trouble.


In the future, carefully considering a tech-stack with integration in mind before going to deep into the design stage would be wise.
